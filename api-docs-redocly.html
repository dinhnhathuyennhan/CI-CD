Tạo ticket hỗ trợ mới cho khách hàng

Tạo ticket hỗ trợ mới cho khách hàng trong hệ thống. API yêu cầu user_id hợp lệ và thông tin ticket bao gồm tiêu đề, nội dung, mức độ ưu tiên.

Đóng ticket của khách hàng

API cho phép đóng một ticket cụ thể của khách hàng khi vấn đề đã được giải quyết. Ticket phải tồn tại và thuộc về user_id được chỉ định.
